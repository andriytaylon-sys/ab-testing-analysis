22 доби, очиятити дані та обгрунтувати доцільність внесення змін на широкий загал
Статистичний аналіз та оцінка результатів за допомогою Python(Pandas,Numpy)
Бізнес-контекст: Компанія впровадила оновлений інтерфейс (Варіант B) з метою підвищення конверсії користувачів. Необхідно було оцінити результати експерименту за 22 доби, очистити дані від шумів та математично обґрунтувати доцільність розгортання змін.
 Технологічний стек:
* Python (Pandas, NumPy, Statsmodels)
* Matplotlib, Seaborn
* Jupyter Notebook / PyCharm
 Виконана робота:
1. Валідація та чищення даних: Опрацьовано часові мітки, перевірено відсутність дублікатів та технічних помилок спліт-системи (вибірка >19 000 спостережень).
2. Розрахунок метрик: Обчислено конверсію (CR). Зафіксовано абсолютний приріст у тестовій групі на +2.80% (з 6.10% до 8.90%), що дало відносний ліфт метрики на +45.9%.
3. Перевірка гіпотез: Застосовано двовибірковий Z-тест для пропорцій (`proportions_ztest`). Розраховано значення Z-статистики (~7.21) та показник $p\text{-value}$ ($p < 0.05$).
4. Візуалізація: Побудовано графіки розподілу конверсії із відображенням 95% довірчих інтервалів.
Бізнес-результат: нульову гіпотезу відхилено. Захищено рекомендацію про 100% запуск Варіанта B для прогнозованого зростання доходу компанії.
